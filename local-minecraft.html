Minecraft for local kids

(Note - this page is not really private so not putting any names here)

Hi folks! I'm thinking of running a local Minecraft Realms server for our son's friends, this website is for useful links and info.

What is Realms?

Minecraft Realms is Microsoft's private Minecraft servers hosted in the cloud. It runs on their systems and they keep it up to date and secure; it is private, we control it and only people we invite can join.

See https://www.minecraft.net/en-us/realms/realms-tutorial for some detailed information.

There's also the parent's guide to minecraft which is a good basic guide to Minecraft, including a lot of links to settings.

What does it cost

The cost is about £7 a month - and that gives us up to 10 simultaneous players. So £1 a month or maybe just buy me a coffee some time? :)

Note you also may have to pay extra for online access on some systems. On a console you might need one of Xbox Game Pass Core, Nintendo Switch Online, or PlayStation Plus to be able to play online. (Android and iOS and Windows users don't need to pay more)

What do we need to play?

You need a system that can run the current version of Minecraft Bedrock. (Bedrock is the edition used on consoles, phones and tablets) - one of:

  • An Android phone or tablet
  • An iPhone or iPad
  • A Playstation or XBox or Nintendo Switch
  • A Windows laptop

You'll need to own a copy of minecraft for this device!

You also need two Microsoft accounts (which are free) - one for a parent, and one for the child. This is because children need parental permission for online play, even on private servers.

There's an article at https://help.minecraft.net/hc/en-us/articles/4408968616077 with instructions for parents on setting up accounts. In brief:

  • Create a parent account at https://account.microsoft.com
  • Go to the microsoft family homepage
  • If your child already has a microsoft account, you can add them here, otherwise you can create a new account here
  • Share the users' gamertags with me so I can add them to the realm

There are some notes on setting up an account at https://www.minecraft.net/en-us/realms/minecraft-and-microsoft-accounts

What's the plan

From my reading it seems a realm lets us have 3 worlds at once, I was thinking of running:

  • a Creative world for just building stuff - these give every child total power to build things. But they aren't very challenging
  • a Survival world for the more challenging main game

The nice thing about Minecraft is it is mostly cooperative - and kids can play solo when they want to. We will have to make sure they don't destroy each other's creations though.

Initially we can keep the Survival world fairly safe - I might run it as "always day" for a while, as that means no mobs (aka monsters) will appear unless you go somewhere dark. (Monsters in Minecraft only spawn in darkness - an early stage of any game is to get some light sources, and beds to sleep overnight)

Also I'll use a few other "easy" settings - like not losing your stuff if you die. And admins can always 'cheat' and give people support if they need it.

How to administer it

I'm happy to be the main admin - just making sure everything works. But I won't be there all the time - if other folks want to jump in and help keep an eye on things, that'd be great too.

Supervision of things like how much play time parents think is appropriate - I'll leave up to you! Hopefully we can chat on WhatsApp about any admin things.

How to make it fair and fun

This is tricky - these kids are young and still learning how to play well!

I think we'll need:

  • Some basic rules in language a 7yo can understand:
    • Don't break or damage someone else's stuff
    • Don't enter someone else's base without permission (I might give them signs "Rowan's Base - keep out!")
    • Don't hassle or even annoy another player
    • No pranks!
    • Breaking a rule can get you suspended from the server - longer suspensions for worse behaviour
  • I'm not sure if we should discourage them from talking about the game? I don't want it getting cliquey at school.
  • Maybe we need to do a trial and keep talking about what can be improved
