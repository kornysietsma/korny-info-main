Minecraft for local kids

(Note - this page is not really private so not putting any names here)

Hi folks! I'm thinking of running a private online Minecraft game for our son's friends, this website is for useful links and info.

What's the plan

This is open for discussion but my thoughts are:

I can run an online game for the kids to play together, with some supervision. It would run on a private server - Microsoft will rent you a private server for £7 a month, so the cost is pretty small (you might need to also pay for online gaming from a console - see below)

That pays for two games so I thought we'd run:

  • a Creative world for just building stuff - these give every child total power to build things. But they aren't very challenging
  • a Survival world for the more challenging main game - though I'd set it up with fairly easy settings.

Probably we'd start with kids coming to our place (with their devices!) so we can show them how it works, get everything set up, and the rest.

The nice thing about Minecraft is it is mostly cooperative - and kids can play solo when they want to. We will have to make sure they don't destroy each other's creations though.

How to make it fair and fun

This is tricky - these kids are young and still learning how to play well!

I think we'll need:

  • Some basic rules in language a 7yo can understand:
    • Play nicely, don't be mean
    • Don't break or damage someone else's stuff
    • Don't enter someone else's base without permission (I might set up some separate private areas before they play)
    • No pranks!
    • Breaking a rule can get you suspended from the server - longer suspensions for worse behaviour
  • An introduction session - so the kids can be helped getting started and play together, and they can talk to each other (the game has chat but only text messages)
  • We can discuss on Whatsapp how things are going and tweak as we go

What do we need to play?

You need a copy of Minecraft (Bedrock edition, which is the standard on consoles and mobile devices)

On a console you might need to pay for online access - Nintendo Switch Online, PlayStation Plus or Xbox Game Pass Core (Android and Apple phone/tablet users don't need to pay more)

To play online you also you need a free Microsoft account. You actually need two Microsoft accounts - one for a parent, and one for the child. This is because children need parental permission for online play, even on private servers.

There's an article at https://help.minecraft.net/hc/en-us/articles/4408968616077 with instructions for parents on setting up accounts. In brief:

  • Create a parent account at https://account.microsoft.com
  • Go to the microsoft family homepage
  • If your child already has a microsoft account, you can add them here, otherwise you can create a new account here
  • You should get a 'gamertag' - you can share the users' gamertags with me so I can add them to the realm

There are some notes on setting up an account at https://www.minecraft.net/en-us/realms/minecraft-and-microsoft-accounts
